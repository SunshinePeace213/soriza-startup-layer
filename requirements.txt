# Python dependencies for this project, including the prompt-architect skill's
# bundled scripts (.claude/skills/prompt-architect/scripts/).
# Centralized at the project root so all Python deps live in one place.
#
# There are NO required third-party packages:
#   * Validation (scripts/quick_validate.py) runs on the Python 3 standard library.
#   * The eval / description-optimization scripts (run_eval.py, run_loop.py,
#     improve_description.py) call the `claude` CLI as a subprocess — not the
#     Anthropic SDK — so there is no `anthropic` dependency.
#
# The only external requirement is the `claude` CLI on PATH, used by the Phase 5/6
# eval scripts. That is a CLI tool, not a pip package.
#
# OPTIONAL — uncomment to harden YAML frontmatter parsing in quick_validate.py.
# The validator falls back to a stdlib parser when this is absent, so it is a
# convenience, not a requirement:
# PyYAML>=6.0

# ---------------------------------------------------------------------------
# Running with uv (no manual venv, no global installs):
#
#   # validate an artifact — zero install, stdlib only:
#   uv run python -m scripts.quick_validate <artifact-path> --artifact-type subagent
#
#   # with the optional PyYAML hardening, in an ephemeral env:
#   uv run --with pyyaml python -m scripts.quick_validate <artifact-path>
#
#   # if you do want a persistent env from this file:
#   uv pip install -r requirements.txt
# ---------------------------------------------------------------------------
